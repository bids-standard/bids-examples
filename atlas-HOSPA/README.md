# Harvard-Oxford Subcortical Probabilistic Atlas

This repository contains the Harvard-Oxford Subcortical Probabilistic Atlas (HOSPA) for the MNI152NLin6Asym and MNI152NLin2009cAsym templates,
organized in BIDS format.

The HOSPA atlas files were downloaded from TemplateFlow and do not match the files in https://github.com/HOA-2/SubcorticalParcellations.
